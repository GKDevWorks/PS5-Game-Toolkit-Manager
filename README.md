# PS5 Game Toolkit & Manager

A Windows desktop application for managing and converting PS5 game
archives, exFAT images and extracted game folders.

## Supported Conversions

- `.ffpfsc` → `.exfat`
- `.ffpfsc` → Folder
- `.exfat` → Folder
- `.exfat` → `.ffpfsc`
- Folder → `.exfat`
- Folder → `.ffpfsc`

## Main Features

- Six dedicated conversion pages
- Batch input processing
- File Scanner for Source and Output folders
- PS5 game metadata and icon detection
- Conversion progress, speed and estimated remaining time
- Input file and folder size display
- Automatic unique output naming
- Configurable Source, Temporary and Output folders
- Dark Windows interface
- Single application interface for all conversion operations

## Installation

1. Download the latest RAR package from the Releases section.
2. Extract it to a folder with sufficient free disk space.
3. Run `PS5 Game Toolkit & Manager.exe`.
4. Select the Source, Temporary and Output folders.
5. Choose the required conversion operation.

## Important

- Keep sufficient free disk space available during conversions.
- Do not close the application while an operation is running.
- Test important operations using backup copies of your files.
- This project does not contain or distribute copyrighted game content.
